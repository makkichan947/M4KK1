📂 quux man 目录体系（0–10）:
(0) intro – 系统介绍与全局约定
	系统哲学、手册使用方法

(1) usercmd – 普通用户命令
	ls, cat, m4sh, vicode, f1le, mtop, ivfetch

(2) syscmd – 系统命令与管理工具
	ptdsk, shutdown, mount, umount, ntctl, ntmgr

(3) objlib – 对象库 / API / 编程接口
	object API, runtime 库说明

(4) devel – 开发与构建工具
	langcc, make

(5) pkg – 包管理与软件分发
	pkgmgr, pkgtrib, lever

(6) sysconf – 系统配置文件
	/etc/m4kk1.conf, /pkg/pkgmgr.conf

(7) objfs – 文件系统与存储
	yfs, ext4, btrfs, swap2

(8) osinternals – 内核与系统调用
	y4ku 内核 API, syscalls

(9) net – 网络工具与服务
	ntctl, ntmgr, socket API

(10) meta – 元信息与规范
	DESIGNPPS, naming rules
